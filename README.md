# Forex Inmobiliaria

Landing page responsive para una inmobiliaria ficticia, desarrollada con HTML, CSS y JavaScript. El proyecto presenta propiedades destacadas, servicios inmobiliarios y una sección de contacto profesional.

## Demo

Ver sitio online:
https://franduartedev.github.io/forex-inmobiliaria/

## Características

* Diseño responsive adaptable a escritorio y dispositivos móviles.
* Hero principal con llamada a la acción.
* Cards de propiedades con imagen, ubicación, precio y detalles.
* Sección de servicios inmobiliarios.
* Formulario de contacto con información comercial.
* Menú de navegación por secciones.
* Estilos organizados con variables CSS.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts

## Secciones del sitio

* Inicio
* Propiedades
* Servicios
* Contacto

## Objetivo del proyecto

El objetivo de este proyecto es practicar maquetación web, diseño responsive, estructura semántica y presentación visual de una landing page orientada a un negocio real.
